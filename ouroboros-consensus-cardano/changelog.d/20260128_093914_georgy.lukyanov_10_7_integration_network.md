### Breaking

- Implement `txWireSize` method for Byron and Shelley `TxLimits` instantiations.

- Add golden test files for `CardanoNodeToClientVersion19` (N2C19) protocol version,
  covering all Cardano eras (Byron, Shelley, Allegra, Mary, Alonzo, Babbage, Conway, Dijkstra).

- Add `FromJSON` and `ToJSON` instances for `ShelleyHash` to support JSON serialization.

- Update `GetBigLedgerPeerSnapshot` query to include a `Proxy` parameter in `SomeLedgerPeerSnapshot`.

- Integrate `ouroboros-network-0.23` and `ouroboros-network-0.24` package reorganization.

- Integrate `typed-protocols-1.1` changes to serialization code.

- Update ImmDBServer to use the new network API.
